# Manga Dashboard (漫画风手搓导航主页)

这是一个使用 **Vue 3 + TypeScript + Vite + Tailwind CSS v4** 从零手搓的个人导航与小工具主页。它拥有和 `sun-panel` 相同的核心功能，并融合了极具视觉张力的 **Neobrutalism（新野兽主义/漫画风）** 设计风格。

## 核心设计与功能

1. **漫画风 Neobrutalism 视觉艺术**：
   - 粗黑边框（`border-4 border-black`）配合直角无羽化纯黑硬阴影（`shadow-[4px_4px_0px_#000]`）。
   - 主页支持 **半色调网点 (Halftone Dots)**、**经典网格线 (Grid)** 以及**纸纹纯色**三档背景图样切换。
   - 引入手绘手写体英文字体与圆润卡通中文字体，标题与卡片支持微小的斜角倾斜。
   - 鼠标悬浮时，卡片向左上方移动并伴随投影变厚，点击按钮时向右下方按下。

2. **动态小工具注册系统 (Widget Registry)**：
   - 项目内置了 5 款功能健全的小工具（位于 `src/components/widgets`）：
     - 🕒 **漫画时钟**：显示当前实时时间、日期与星期。
     - 🔍 **超级搜索栏**：支持 Google、百度、必应三引擎切换搜索。
     - 💻 **性能监控器**：模拟 CPU、内存、磁盘的动态负载进度条。
     - 📝 **漫画任务清单**：支持新增、完成勾选、删除任务。
     - ✏️ **随手小记**：支持多行文字备忘录且支持防抖自动保存。
   - **如何新增小工具**：只需在 `src/components/widgets` 下新建您的 Vue 组件，然后在 `src/components/WidgetRegistry.ts` 中声明它即可，主页面会自动读取并动态加载，极易扩展。

3. **导航与分组管理**：
   - 支持动态添加、修改、删除导航分组，分组名称卡片支持自定义命名。
   - 支持在各分组下动态添加、修改、删除导航卡片。卡片支持自定义标题、URL 链接、描述说明、预置图标（Github, Home, Video, Search, Link）以及六种明快粉嫩的卡片底色。

4. **配置中心与本地持久化**：
   - 点击右上角设置图标即可打开配置中心。
   - 可在配置中心修改：Logo 文本、背景遮罩网纹类型、开启/关闭漫画手写字体、以及**独立开关每个已注册的小工具**。
   - 所有导航数据（Groups, Items）与配置数据（Config）均通过 `LocalStorage` **全自动持久化在您的浏览器中**，无需安装任何数据库或后端，启动即用！

## 本地开发与启动

### 环境准备
确保您的电脑上安装了 [Node.js](https://nodejs.org) 并安装了 `pnpm`。

### 安装依赖
在 `d:\code\manga-dashboard` 目录下运行命令：
```powershell
pnpm install
```

### 启动开发服务器
运行以下命令启动项目：
```powershell
pnpm run dev
```
启动后，在浏览器中打开命令行输出的地址（通常为 `http://localhost:5173`）即可进行预览和开发。

### 编译打包
如需编译生成静态网页，运行：
```powershell
pnpm run build
```
编译产物会输出到 `dist` 目录下，您可以直接将 `dist` 里的静态文件托管到 Nginx、Caddy、Vercel 或您的个人服务器中，即可完成部署！
